# Sequential Calculator

Sequential Calculator is a simple Android application that performs arithmetic
calculations in a step-by-step sequence rather than complex expressions.

The app is designed for clarity, learning, and controlled calculation workflows.

## Features
- Sequential arithmetic operations
- Minimal and clean UI
- Offline functionality
- No data collection
- Lightweight and fast

## Use Cases
- Educational purposes
- Verifying stepwise calculations
- Simple arithmetic without scientific clutter

## Privacy
This app does not collect, store, or share personal user data.
A full privacy policy is available publicly.
